Penafian

Adalah dimaklumkan bahawa perisian dan aplikasi web ini merupakan panduan sahaja dalam pengiraan kecekapan saiz tangki untuk penuaian air hujan. NAHRIM tidak bertanggungjawab terhadap sebarang kehilangan atau kerugian yang dialami oleh pengguna disebabkan penggunaan keputusan perisian dan aplikasi web.

Notis Privasi

Laman web kami menggunakan Google Analytics untuk mengumpulkan maklumat statistik tertentu secara automatik seperti bilangan dan kekerapan pelawat serta alamat IP mereka. Maklumat ini digunakan sebagai maklumat statistik mengenai pengguna, mengikut alamat IP. Kami tidak mengenal pasti pengguna individu daripada data ini. Maklumat ini membantu kami untuk melihat cara pengguna menggunakan tapak web dan perkhidmatan kami, supaya kami boleh terus memperbaiki perkhidmatan kami. Maklumat tentang penggunaan laman web anda secara amnya akan dihantar dan disimpan oleh Google dalam pelayan di Amerika Syarikat.

Untuk maklumat lebih lanjut mengenai maklumat yang dikumpulkan menggunakan Google Analytics sila lawati: https://policies.google.com/privacy

Anda boleh memilih untuk tidak menyertai Google Analytics di browser anda dengan memuat turun add on di sini: http://tools.google.com/dlpage/gaoptout
